# Compra Inteligente

https://matheus-f-reis.github.io/compra-inteligente/

Projeto desenvolvido para a disciplina Construção de Páginas Web II do Instituto Federal do Sudeste de Minas Gerais.

## Objetivo

Este sistema ajuda o usuário a decidir se vale a pena comprar um produto agora ou esperar.

Para isso, o sistema:

- pesquisa informações recentes utilizando a API Serper;
- envia essas informações juntamente com a pesquisa do usuário para a API Gemini;
- recebe uma resposta em formato JSON;
- apresenta o resultado em uma interface web.

## Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript
- API Serper (Google Search)
- API Gemini

## Como executar

1. Faça o download do projeto.
2. Abra o arquivo `index.html` em um navegador.
